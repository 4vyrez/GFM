# Cat Images for CaptchaCats Game

This folder contains images for the two-phase Captcha game.

## Structure

```
cats/
├── luzi/           # Your cat Luzi's photos
│   ├── luzi_1.jpg
│   ├── luzi_2.jpg
│   └── luzi_3.jpg
├── toni/           # Your cat Toni's photos
│   ├── toni_1.jpg
│   ├── toni_2.jpg
│   └── toni_3.jpg
└── distractors/    # Other animals (optional - emoji fallback works!)
    ├── dog_1.jpg
    ├── bunny_1.jpg
    └── ...
```

## How to Add Photos

1. **Luzi's photos**: Add 3-4 photos of Luzi to the `luzi/` folder
   - Name them: `luzi_1.jpg`, `luzi_2.jpg`, `luzi_3.jpg`, etc.
   - Best: clear face shots, different angles

2. **Toni's photos**: Add 3-4 photos of Toni to the `toni/` folder
   - Name them: `toni_1.jpg`, `toni_2.jpg`, `toni_3.jpg`, etc.
   - Best: clear face shots, different angles

3. **Distractor animals** (optional): Add other animal photos to `distractors/`
   - If you don't add any, the game will use emoji animals as fallback
   - Good options: dogs, bunnies, hamsters, etc.

## Image Requirements

- **Format**: JPG or PNG
- **Size**: Square-ish images work best (they'll be cropped to squares)
- **Resolution**: At least 200x200px recommended

## After Adding Photos

The game will automatically detect and use any images you add!
